Deep Sea Diver

This game was developed in Unity for the 2021 Ludum Dare Game Jam. This was my first Game Jam and I collaborated with my brothers to build this game. I was responsible for all sound design and programming, while they handled art and level design. This game is a short 2D platformer where you control a diver trying to get to the bottom of the ocean. You can swim and throw a trident, and you must collect air to stay alive. Your air supply depletes over time and enemies can knock extra air out of you.

Snake

This game runs a text-based version of the classic Snake Game in a Windows console terminal. The snake is controlled with the arrow keys. The snake grows longer and moves faster as you collect food. The game also saves the ten highest scores in a .txt file, which it reads and displays when first starting the application. See the source code for downloadable a version of the game.

JRPG Battle Simulator

This project is the first game I've created. This game is completely text-based and runs in a Windows console terminal. This game repeatedly presents you with enemies that scale in difficulty as a you progress, and you are given a list of actions to defeat enemies. You can attack, use magic, or use items to fight. After every enemy, you are given the opportunity to buy upgrades or items for use in future fights. There is also a basic elemental system where you can deal extra damage to enemies by attacking them with an element-based attack that they are weak to. See the source code for a downloadable version of the game.

Rocket Golf

This project is currently a work in progress and is being developed in Unity. It is a top-down 2D shooter where you control a "golf ball" that moves by shooting a gun and using its recoil. Your goal is to progress through levels without dying. You must kill enemies, dodge hazards, destroy walls, and collect upgrades. This project will eventually be playable on Android and in browsers online. View the project on Itch.io to see the current progress of the game and read the devlogs.
